# 🎬 Pinpol Eventos - Portafolio Interactivo 🚀

¡Bienvenido al repositorio oficial del Portafolio Interactivo de **Pinpol Eventos**! Una plataforma web moderna, rápida y visualmente impactante diseñada para mostrar servicios de producción audiovisual de alta gama en Lima, Perú.

## ✨ Características Principales

Este proyecto no es solo una galería; es una experiencia inmersiva optimizada para la conversión:

* **🖼️ Galería Multimedia Inteligente**: Soporte nativo para imágenes y videos de alta definición con un sistema de Lightbox personalizado.
* **♾️ Reel de Clientes Infinito**: Sección dinámica de marcas que confían en la productora con efecto de desplazamiento suave.
* **📱 Ultra Responsive**: Optimizado específicamente para dispositivos de alta gama (como el **Samsung Galaxy S23 Ultra**) asegurando que la navegación y el contenido se centren perfectamente en cualquier pantalla.
* **⚡ Filtrado en Tiempo Real**: Sistema de categorías para explorar servicios de Streaming, Karaoke, Sonido, Pantallas, Drones e Iluminación de forma instantánea.
* **🟢 Call-to-Action Directo**: Integración estratégica con WhatsApp para presupuestos express.

---

## 🛠️ Stack Tecnológico

* **Frontend**: [React.js](https://reactjs.org/)
* **Estilos**: [Tailwind CSS](https://tailwindcss.com/) (Mobile-first design)
* **Iconografía**: [Lucide React](https://lucide.dev/)
* **Animaciones**: CSS keyframes optimizados para GPU.
* **Despliegue**: [Vercel](https://vercel.com/) / GitHub Pages.

## 📂 Estructura del Proyecto

```text

````

## 🚀 Instalación y Uso Local

1.  **Clonar el repositorio:**

    ```bash
    git clone [https://github.com/acastroascencio/pinpol-portfolio.git](https://github.com/acastroascencio/pinpol-portfolio.git)
    ```

2.  **Instalar dependencias:**

    ```bash
    npm install
    ```

3.  **Iniciar el servidor de desarrollo:**

    ```bash
    npm run dev
    ```

4.  **Abrir en el navegador:**
    Visita `http://localhost:5173` (El puerto depende de tu configuración)
